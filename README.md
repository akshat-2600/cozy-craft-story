# COZY DECORS

COZY DECORS is a small e-commerce demo showcasing handmade crafts like explosion boxes, greeting cards, and 3D albums.

## Quick start

1. Install dependencies:

```sh
npm install
```

2. Run the development server:

```sh
npm run dev
```

3. Build for production:

```sh
npm run build
```

## Technologies

- Vite
- TypeScript
- React
- Tailwind CSS
